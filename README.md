# AMD-NG2-UI #

This project is an Angular 2 UI Component Library.

## Features ##

* Native Angular2 DOM manipulation
* Pure CSS driven animation
* Customisable
* Themeable

## Setup ##

* [Install NodeJS](https://nodejs.org)
* Clone the repository locally
* Install the project's dependencies
``` bash
$ npm install
```
* Run the project
```
$ npm start
```
